# Restrições

- Não usar tokens transferíveis como ERC-20/ERC-721 para representar votos.
- Não publicar escolhas eleitorais em claro.
- Não gerar recibos que provem a opção escolhida pelo eleitor.
- Não guardar IP, identidade civil e boletim no mesmo domínio de dados.
- Não permitir apuramento dependente de uma única chave privada individual.
- Não usar criptografia artesanal em produção.
- Não usar modelos de IA para aceitar, rejeitar ou inferir votos.
