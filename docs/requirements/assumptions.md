# Pressupostos

1. O projecto é académico e experimental.
2. O token é uma credencial anónima de elegibilidade, não uma moeda.
3. O voto é cifrado no cliente antes da submissão.
4. A autoridade de identidade não tem acesso ao boletim.
5. A urna digital não conhece a identidade civil do eleitor.
6. Observadores externos conseguem verificar o bulletin board.
7. A demo local não oferece unlinkability de produção.
8. Criptografia educativa serve para compreensão, não para implantação real.
