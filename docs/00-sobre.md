# 00 — Sobre o projecto

O **Civitas Vote Token** propõe uma arquitectura de voto digital baseada num conceito simples: separar a prova de elegibilidade da escolha eleitoral.

Num sistema de voto presencial, esta separação é intuitiva: a mesa confirma que o eleitor pode votar, entrega um boletim, o eleitor vota em segredo e deposita o boletim na urna. Num sistema digital, esta separação tem de ser reconstruída por meios criptográficos e organizacionais.

O projecto usa o termo **Vote Token** para designar uma credencial anónima, emitida a um eleitor elegível, que permite depositar exactamente um boletim cifrado.

## O que este projecto é

- Um projecto técnico para GitHub.
- Uma arquitectura documentada.
- Um demonstrador de ciclo de voto.
- Uma base académica para estudar voto electrónico.
- Um ponto de partida para investigação em credenciais anónimas, bulletin boards e verificação pública.

## O que este projecto não é

- Não é uma solução pronta para eleições oficiais.
- Não é uma moeda.
- Não é um token ERC-20.
- Não é uma NFT.
- Não é uma desculpa para meter blockchain em tudo como se fosse molho de francesinha tecnológica.

## Tese do projecto

Um sistema de voto digital minimamente sério deve garantir simultaneamente:

1. elegibilidade;
2. unicidade do voto;
3. segredo do voto;
4. integridade do apuramento;
5. auditabilidade pública;
6. resistência a coerção, dentro dos limites do voto remoto;
7. acessibilidade e usabilidade.
