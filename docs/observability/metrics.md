# Observabilidade

## Métricas recomendadas

```txt
tokens_issued_total
tokens_redeemed_total
ballots_submitted_total
ballots_rejected_total
duplicate_token_attempts_total
invalid_proof_total
bulletin_board_merkle_root_changes_total
tally_verification_failures_total
audit_findings_open_total
```

## Regra de privacidade

Registar eventos técnicos, não escolhas eleitorais. Nunca correlacionar identidade civil, IP e voto. Observabilidade não deve transformar-se numa câmara de vigilância com gráficos bonitos.
